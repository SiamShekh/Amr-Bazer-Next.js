## Run Locally

Clone the project

```bash
  git clone https://github.com/SiamShekh/Planq-Bangladesh.git
```

Go to the project directory

```bash
  cd Planq-Bangladesh
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
